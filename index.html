What is it ?

Vert.x is a lightweight, high performance application platform for the JVM that's designed for modern mobile, web, and enterprise applications...

What does mean ?

  • simillar to Node.js - but not CLONE !
  • solving C10K problem
  • powered by JVM
  • vertical & horizontal scaling

General

  • Asynchronous by nature
  • High-Performing
  • Slim and leightweight core
  • Polyglot
  • Reusable modules

Core Protocols

  • TCP / UDP
  • HTTP(s) and compression
  • WebSockets / Secure-WebSockets
  • SockJS
  • DNS Client

Polyglot

  • Java
  • JavaScript
  • Groovy
  • Ruby
  • Python
  • Scala*
  • Clojure*
  • PHP*

* under heavy development

History

  • Tim Fox - father of Vert.x
  • VMware - incubator
  • Red Hat - adoptive parent
  • Eclipse Foundation

Thread-Model

  • EventLoops (Usually one per Core)
  • Each Verticle bound to one EventLoop and so single-Threaded
  • No concurrency issues (synchronization, locking, ..)
  • WorkerVerticles for blocking calls (jdbc, http requests, ...)
Never block the EventLoop!
